Tasktracking

As an executive officer,
I want to manage tasks,
so that I can keep track
Scenarios
  • Given an executive officer with no tasks assigned
  • When the officer creates a new task
  • Then ensure it is assigned to the officer
Specifications
  • ExecutiveOfficer
    • Should be able to start a task assigned
    • Should not be able to start a task assigned to others
    • Should only be able to start one task at the time
  • Task
    • Should be created in context of a task list
  • TaskList
    • Should tell if it has tasks in progress
    • Should not have any tasks when new
    • Should not have any tasks in progress when new
Statements
  • ExecutiveOfficer
    • A new executive officer should not have any tasks in his or her task list
